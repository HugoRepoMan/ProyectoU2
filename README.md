# Breaking News - Portal de química

Proyecto React + Vite basado en el laboratorio de componentes reutilizables. La aplicación fue transformada en un portal sencillo de noticias de química llamado **Breaking News**.

## Funcionalidades

- Layout reutilizable con Header, contenido principal y Footer.
- Navegación con React Router.
- Componentes reutilizables con Props y PropTypes.
- CSS Modules para encapsular estilos.
- Página de noticias renderizada con `map()`.
- Calculadora de pH usando `useState`, eventos y funciones matemáticas.
- Buscador de compuestos químicos usando Fetch API y PubChem.
- Pruebas unitarias con Vitest y React Testing Library.

## Instalación

```bash
npm install
npm run dev
```

## Pruebas

```bash
npm run test:run
npm run coverage
```

## Producción

```bash
npm run build
```

## Rutas

- `/` Inicio
- `/noticias` Noticias de química
- `/calculadora` Calculadora de pH
- `/compuestos` Buscador de compuestos con PubChem
